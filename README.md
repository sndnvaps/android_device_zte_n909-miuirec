此项目托管的文件是用于编译 Miui Recovery

源代码是用CM10.1


下载的源代码请放置到/源代码目录/device/ZTE/N909
如果device/ZTE/N909不存在，请创建一个。
```
git clone git@github.com:sndnvaps/android_device_zte_n909-miuirec.git -b cm-10.1 /源代码目录/device/ZTE/N909 
```
